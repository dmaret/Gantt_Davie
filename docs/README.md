# Documentation

Organisation des documents par thème :

- **`security/`** — Audits de sécurité, rapports de vulnérabilités
- **`setup/`** — Guides d'installation, configuration initiale, autonomie de l'app
- **`architecture/`** — Architecture système, conception technique, diagrammes
- **`releases/`** — Notes de version, changelogs, historique des releases
